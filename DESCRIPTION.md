## 1. 프로젝트 목적

발리 여행 블로그를 **자동 생성**하여 네이버 AI 검색(Cue:)에 최적화된 콘텐츠를 대량으로 배포합니다. 마이리얼트립 제휴 링크를 통한 수익화를 목표로 합니다.

### 핵심 가치 제안
- **11개 발리 지역** × **6개 카테고리** × **14개 페이지** = **924개 고유 HTML**
- **8,926개 고유 이미지** (지역/카테고리별 분류)
- **네이버 SEO 최적화**: 구조화된 데이터, 메타 태그, canonical URL
- **수익화**: 마이리얼트립 할인쿠폰 + 투어/호텔 제휴 링크

## 4. 페이지 구조 (HTML)

각 HTML 페이지는 다음 섹션으로 구성됩니다:

```
┌─────────────────────────────────────┐
│ 상단 파트너스 고지문                    │
├─────────────────────────────────────┤
│ 헤더 (제목, 카테고리, 날짜, 작성자)     │
├─────────────────────────────────────┤
│ 브레드크럼 (홈 > 지역 > 카테고리)       │
├─────────────────────────────────────┤
│ Q&A 서론 (페이지별 변형)               │
│ 핵심 팁 요약 박스                      │
│ 쿠폰 이미지 + 하이퍼링크 (상단)         │
├─────────────────────────────────────┤
│ 이미지 10장 (페이지별 고유 배분)        │
├─────────────────────────────────────┤
│ 카테고리별 콘텐츠                      │
│  - 맛집/명소 리스트 (이름, 가격, 팁)   │
│  - 가격비교표 (로컬 vs 투어 vs 호텔)   │
│  - 추천 명소                          │
│  - 숨은 명소                          │
├─────────────────────────────────────┤
│ 교통 정보                             │
│ 숙소 추천 (예산/중급/고급)             │
│ 실전 팁 (카테고리별 3개)              │
│ 관련 지역 링크                        │
├─────────────────────────────────────┤
│ 쿠폰 이미지 + 하이퍼링크 (하단)         │
│ 태그                                 │
├─────────────────────────────────────┤
│ 푸터 (저작권, 파트너스 고지)            │
└─────────────────────────────────────┘
```

## 6. SEO 최적화

### 6.1 메타 태그
- `<title>`: 지역 + 카테고리 + 페이지번호 + 연도
- `<meta description>`: 150자 이내, 핵심 키워드
- `<link rel="canonical">`: 중복 URL 방지
- `<meta property="og:*">`: Open Graph (Facebook/카카오)

### 6.2 구조화된 데이터 (JSON-LD)
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "우붓 음식/맛집 여행 가이드 — 2026 최신",
  "author": {"@type": "Person", "name": "발리 여행 10년차 블로거"},
  "datePublished": "2026-04-01"
}
```

### 6.3 내부 링크
- 각 페이지에서 같은 카테고리의 다른 지역 3개를 랜덤 링크
- 브레드크럼: 홈 > 지역 > 카테고리 > 현재 페이지

## 8. 콘텐츠 데이터

### 8.1 데이터 소스 (`BALI_DATA`)
- 11개 지역별 상세 데이터 (딕셔너리)
- 각 지역당: Q&A, 맛집(6곳), 문화(2~4곳), 해변(2~4곳), 자연(2~4곳), 쇼핑(2~3곳), 교통(2~3곳)

- 실전 팁: 10개 중 3개 선택 (시작점 변형)
- 관련 지역: 11개 중 3개 랜덤 선택
