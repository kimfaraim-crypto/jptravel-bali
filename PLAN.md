## 1. 프로젝트 비전

발리 여행 관련 **네이버 AI 검색(Cue:) 최적화 콘텐츠**를 자동으로 대량 생성하여, 검색 상위 노출을 통한 유기적 트래픽 확보와 마이리얼트립 제휴 수익화를 달성합니다.

## 4. 다음 단계 계획

### Phase 4: 콘텐츠 품질 향상 (우선순위: 높음)

#### 4.1 실제 발리 이미지 크롤링
- **목적**: Picsum 랜덤 이미지 → 실제 발리 사진으로 교체
- **방법**: Openverse API + Flickr에서 "bali temple", "bali food" 등 키워드 검색
- **예상 일정**: 1~2일
- **예상 결과**: 카테고리별 실제 발리 사진 50개+/조합

- **예상 일정**: 2~3일
