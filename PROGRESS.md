# JP Travel Bali — 발행 여행 블로그 프로젝트

## 프로젝트 개요
- **924개 고유 HTML 페이지** (11지역 × 6카테고리 × 14페이지)
- **다중 소스 이미지 시스템** (Openverse + Flickr + Wikimedia + Picsum)
- **네이버 AI 검색(Cue:) 최적화** 구조
- **마이리얼트립 제휴 링크** 시스템

## 지역 (11개)
우붓, 스미냑, 꾸따, 사누르, 누사두아, 울루와뚜, 짠디다사, 로비나, 킨타마니, 타나롯, 베두굴

## 카테고리 (6개)
- 🍜 음식/맛집 (food)
- 🛕 문화/사원 (culture)
- 🏖️ 해변/서핑 (beach)
- 🌿 자연/모험 (nature)
- 🛍️ 쇼핑/마사지 (shopping)
- 🚗 여행/교통 (transport)

## 빌드 방법

```bash
# 1. 이미지 스크래핑 (4소스: Openverse + Flickr + Wikimedia + Picsum)
python3 scraper_enhanced.py

# 2. HTML 빌드 (924개 고유 페이지 생성)
python3 build_bali_v5.py
```

## 📝 기술 노트

### 이미지 소스 API 정보
| 소스 | API | Rate Limit | 비고 |
|------|-----|------------|------|
| Openverse | `api.openverse.org/v1/images/` | 2초 대기 | 결과不稳定, 간헐적 0개 |
| Flickr | `flickr.com/services/feeds/photos_public.gne` | 0.5초 대기 | 안정적, 20개/쿼리 |
| Wikimedia | `commons.wikimedia.org/w/api.php` | 2초 대기 | User-Agent 필수 |
| Picsum | `picsum.photos/seed/{seed}/1200/800` | 제한 없음 | 랜덤 이미지 (비발리) |

## ⚠️ 알려진 이슈
1. **Openverse API 불안정**: 간헐적으로 0개 결과 반환, 재시도 필요
2. **이미지 다운로드 실패율**: 약 50~60% (URL 만료, 404, 타임아웃)
3. **Picsum fallback**: 발리 실제 사진이 아닌 랜덤 이미지 → 실전 배포 전 교체 필요
4. **Wikimedia URL 생성**: MD5 해시 기반 URL 생성 방식, 일부 파일명에서 실패
5. **콘텐츠 중복도**: 템플릿 변형 방식이라 같은 조합 내 유사도 약 30~40%
