1. 네이버 AI 검색(Cue:) 최적화 전략
============================================================

1-1. 핵심 원칙
  "AI에게 인용되는 글은 잘 쓴 글이 아니라, 잘 쪼개진 글이다."

  인용 확률 ≈ (구체 사실 밀도) × (구조 분리도) × (질의-제목 일치도) × (신선도)
                ÷ (필러 분량)

1-2. 현재 적용 상태
  ✅ Q&A 형식 서론 (질문→답변)
  ✅ 가격비교 표 (로컬/투어/호텔)
  ✅ 구체적 가격 (루피아 + 원화)
  ✅ 번호 리스트 (명소 TOP5)
  ✅ 소제목에 키워드 포함
  ✅ JSON-LD 구조화 데이터
  ✅ 동적 datePublished

1-3. 추가 개선 필요
  ⚠️ 한 단락 = one claim 원칙 강화
  ⚠️ FAQ를 지역별 고유 질문으로 교체
  ⚠️ "YYYY년 MM월 기준" 명시적 날짜 삽입
  ⚠️ 구체 식별자(쿠폰 코드, 상품명) 추가 노출

2. 키워드 전략
============================================================

2-1. 지역별 핵심 키워드 (검색량 기준)

  우붓: 우붓 맛집, 우붓 라이스테라스, 우붓 원숭이숲
  스미냑: 스미냑 비치클럽, 스미냑 선셋, 스미냑 맛집
  꾸따: 꾸따 서핑, 꾸따 비치워크, 꾸따 맛집
  사누르: 사누르 일출, 사누르 자전거, 사누르 맛집
  누사두아: 누사두아 리조트, 누사두아 블로우홀, 누사두아 가족여행
  울루와뚜: 울루와뚜 사원, 울루와뚜 케착춤, 울루와뚜 서핑
  짠디다사: 짠디다사 다이빙, 짠디다사 조용한해변, 짠디다사 베사키사원
  로비나: 로비나 돌고래, 로비나 온천, 로비나 폭포
  킨타마니: 킨타마니 화산, 킨타마니 일출트레킹, 킨타마니 뷔페
  타나롯: 타나롯 사원, 타나롯 일몰, 타나롯 가는법
  베두굴: 베두굴 울룬다누사원, 베두굴 호수, 베두굴 식물원

3. 콘텐츠 전략
============================================================

3-1. 콘텐츠 차별화

  현재 한계:
    - 같은 카테고리 14페이지가 구조가 거의 동일
    - 음식 목록이 고정되어 있어 페이지 간 차별성 부족

  개선 전략:
    a) 테마별 페이지 분화
       - 001~003: 기본 가이드 (입문자용)
       - 004~006: 가성비 추천 (예산 여행)
       - 007~009: 프리미엄 추천 (럭셔리 여행)
       - 010~012: 숨은 명소 (현지인 추천)
       - 013~014: 시즌별 추천 (우기/건기)

    b) 콘텐츠 깊이 확장
       - 음식: 15~20개 레스토랑 데이터, 페이지별 4개 선택
       - 명소: 입장료, 영업시간, 소요시간, 난이도 추가
       - 교통: 시간대별 요금 변동, 대안 경로 제시

3-2. 신선도(Freshness) 전략

  - 모든 페이지에 "2026년 X월 기준" 명시
  - 가격 변동 시 본문 상단에 "[YYYY-MM-DD 업데이트]" 박스
  - 분기 1회 전면 가격 업데이트
  - 댓글/공감 활동으로 마지막 활동 시점 갱신

3-3. 내부 링크 전략

  현재: 관련 지역 추천 3개
  개선:
    - 같은 지역 다른 카테고리 링크 (예: 우붓 맛집 → 우붓 사원)
    - 같은 카테고리 다른 지역 링크 (예: 우붓 맛집 → 꾸따 맛집)
    - "이 글과 비슷한 글" 추천 (5개)
    - 시리즈 글 연결 (예: 발리 자유여행 시리즈 1/2/3)

4. 기술적 SEO
============================================================

4-1. 메타태그 최적화
  ✅ title 태그 (고유, 키워드 포함)
  ✅ meta description (고유, CTR 최적화)
  ✅ meta keywords
  ✅ canonical URL
  ✅ og:title, og:description, og:image
  ✅ twitter:card
  ✅ JSON-LD Article 스키마

4-2. 추가 필요
  ⚠️ sitemap.xml (924개 URL)
  ⚠️ robots.txt
  ⚠️ hreflang 태그 (한국어)
  ⚠️ favicon.ico
  ⚠️ 404 에러 페이지

4-3. 모바일 최적화
  ✅ 반응형 뷰포트
  ✅ 모바일 폰트 크기
  ✅ 터치 친화적 버튼
  ✅ 이미지 지연 로딩 (loading="lazy")

5. 링크 빌딩 전략
============================================================

5-1. 제휴 링크 전략
  현재: 마이리얼트립 파트너스
  - 쿠폰 이미지 상단+하단 2회 노출
  - 투어/티켓 링크 자연스럽게 본문 삽입
  - rel="noopener sponsored" 적용 (완료)

  주의사항:
  - 페이지당 6~8개 이내 유지
  - 1px 숨겨진 링크 제거 (완료)
  - 파트너스 고지문 상단 배치 (완료)

5-2. 외부 링크 전략
  - 네이버 블로그 시리즈 글로 유입
  - 카카오맵 장소 등록
  - 인스타그램 해시태그 (#발리여행 #우붓맛집 등)
  - YouTube 여행 영상 설명란에 링크
