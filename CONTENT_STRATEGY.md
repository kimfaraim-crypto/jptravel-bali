## 1. 콘텐츠 생성 전략

### 1.1 핵심 원칙
1. **고유성**: 924페이지 전부 다른 콘텐츠
2. **전문성**: 10년차 블로거 관점의 실전 정보
3. **가격 정보**: 모든 장소에 구체적 가격 (Rp + 원화)
4. **실용성**: 예산 가이드, 추천 일정, 교통 정보 포함
5. **SEO 최적화**: 네이버 AI 검색(Cue:) 최적화

### 1.2 콘텐츠 구조 (모든 페이지 공통)
```
[1] 도입부 (Hook) — 독자의 관심을 끄는 첫 문장
[2] 쿠폰 이미지 — 마이리얼트립 제휴 링크
[3] 이미지 10장 — 장소별 사진 + figcaption
[4] Q&A 4개 — 자주 묻는 질문 + 고유 답변
[5] 추천 장소 5개 — 가격/위치/팁/추천메뉴/블로거팁
[6] 가격 비교표 — 로컬 vs 투어 vs 호텔
[7] 추천 명소 — 5개 장소 리스트
[8] 숨은 명소 — 1개 (지역별 고유)
[9] 교통 정보표 — 공항→지역, 시내 이동
[10] 숙소 추천표 — 예산형/중급/고급
[11] 예산 가이드 — 1박 2일 예산 + 추천 일정
[12] 현지인 꿀팁 — 1개 (지역별 고유)
[13] 실전 팁 — 5개 (지역+카테고리별 고유)
[14] 여행 에피소드 — 1개 (지역별 고유 경험담)
[15] 관련 지역 링크 — 3개
[16] 태그 — 6개
[17] 마무리 — 업데이트 날짜
```

## 3. Q&A 전략

### 3.1 Q&A 구조
```html
<h2>❓ 자주 묻는 질문</h2>
<div style="...">
    <p style="...">❓ {질문}</p>
    <p style="...">{답변}</p>
</div>
```

### 3.2 질문 템플릿 (카테고리별)
| 카테고리 | 질문 예시 |
|----------|-----------|
| food | {지역} 맛집 중 가장 추천하는 곳은? |
| food | {지역} 가성비 맛집은 어디인가요? |
| food | {지역}에서 꼭 먹어봐야 할 음식은? |
| culture | {지역} 사원 방문 시 복장 규정은? |
| culture | {지역}에서 꼭 봐야 할 전통 공연은? |
| beach | {지역} 해변에서 서핑 강습 받으려면? |
| beach | {지역} 비치클럽 추천은? |
| nature | {지역} 라이스 테라스 방문 팁은? |
| nature | {지역} 폭포 추천 및 가는 방법은? |
| shopping | {지역} 기념품 쇼핑 추천 장소는? |
| shopping | {지역} 마사지 가격과 추천 샵은? |
| transport | {지역} 공항에서 가는 방법은? |
| transport | {지역} 시내 이동 수단 추천은? |

- 모든 답변에 구체적 가격 포함 (Rp + 원화)
- 같은 페이지 내 답변 중복 없음
- 11지역 × 6카테고리 = 66개 조합 모두 고유 답변

## 4. 블로거 팁 전략

### 4.1 팁 구조
```html
<p style="..."><strong>✍️ 블로거 팁:</strong> {팁 내용}</p>
```

### 4.2 팁 기준
- 모든 추천 장소에 블로거 팁 포함
- 같은 페이지 내 팁 중복 없음
- 지역별 고유 팁 (8개 이상 보유)
- 구체적이고 실용적인 정보

### 4.3 팁 예시 (우붓)
```
- 우붓 아트 마켓은 오전 8시 전에 가야 진짜 좋은 물건을 구할 수 있어요.
- 우붓 라이스 테라스는 우기(11~3월)에 녹색이 가장 예뻐요.
- 우붓 원숭이 숲에서 선글라스·모자·핸드폰 꼭 잡으세요.
- 우붓 마사지는 시내보다 외곽이 30~50% 저렴해요.
```

## 6. 가격 전략

### 6.1 가격 표기 형식
```
{Rp 금액} (약 {원화 금액}원)

예시:
- 25,000Rp (약 2,500원)
- 150,000Rp (약 15,000원)
- 500,000Rp (약 50,000원)
```

### 6.2 환율 기준
```
1 Rp ≈ 0.1원 (대략적 환율)
25,000Rp = 2,500원
100,000Rp = 10,000원
```

### 6.3 가격 비교표 구조
```html
<table>
    <thead>
        <tr>
            <th>항목</th>
            <th>💰 로컬</th>
            <th>🎫 투어</th>
            <th>🏨 호텔</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>나시고렝</td>
            <td>25,000Rp (약 2,500원)</td>
            <td>45,000Rp (약 4,500원)</td>
            <td>80,000Rp (약 8,000원)</td>
        </tr>
    </tbody>
</table>
```
