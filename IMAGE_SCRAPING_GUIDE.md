## 📸 이미지 소스 (우선순위)

### 1차: Flickr 공개 피드 API (현재 사용 중)
- **API**: `https://api.flickr.com/services/feeds/photos_public.gne`
- **키 불필요**, 무료 무제한
- **방식**: 태그 기반 검색 (`bali,ubud,food` 등)
- **이미지 크기**: `_b` 접미사 (1024px)
- **장점**: 안정적, 빠름, 키 불필요
- **단점**: 태그당 20개 제한, 태그 다양화 필요

### 2차: Openverse API (간헐적 차단)
- **API**: `https://api.openverse.org/v1/images/`
- **키 불필요**, 무료
- **소스**: Flickr, Wikimedia, StockSnap 통합
- **주의**: Cloudflare rate limit 간헐적 차단

### 3차: 중국 여행 사이트 (수동 스크래핑)
- **마펑우 (Mafengwo)**: `mafengwo.cn` - 중국 여행 노드/사진
- **큐이얼 (Qyer)**: `qyer.com` - 자유여행 가이드/사진
- **씨트립 (Ctrip)**: `you.ctrip.com` - 여행 일지/사진
- **주의**: JS 렌더링 필요, 모바일 UA 우회 필수

### 4차: 추가 무료 API
- **Pexels**: `api.pexels.com` - 무료 키 발급 필요 (200req/hr)
- **Pixabay**: `pixabay.com/api` - 무료 키 발급 필요 (5000req/day)
- **Unsplash**: `api.unsplash.com` - 무료 키 발급 필요 (50req/hr)

3. 중복 제거 강화 (perceptual hash 임계값 조정)
